# Sunny's NoisyPad
Submitted for Blueprint HackPad.
Features a 4x3 macropad.

## Main Stuff
Mainly used for video editing and CAD designing
* App shortcuts using Windows bindings (taskbar)
* Rotary Encoder to control volume
* OLED Display displaying a cool welcome message

## Layout
|   Save   |    Copy    | Paste | BrightnessUp  |
| Delete  |   Win + 4   | Lock  | BrightnessDn  |
| Undo | Redo | Mute  | Enter |

## Firmware: Used KMK
  * Also added clock/time to OLED display
